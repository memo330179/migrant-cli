## USAGE ##
```
migrant
 
Usage:
  migrant run_migrations
  migrant run_migration <version>
  migrant generate_migration <name>
  migrant -h | --help
  migrant --version
 
Options:
  -h --help                         Show this screen.
  --version                         Show version.
 
Examples:
  migrant -h
 
Help:
  For help using this tool, Ask Guillermo
```
